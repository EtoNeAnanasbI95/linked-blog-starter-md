* Простейший инструмент CI/CD, сделанный на замену старого Travis CI. Имеет схожий синтаксис с GitLab CI, но имеет различие с разделением Джоб на Стадии, а если быть точным, то полное отсуствиие этой удобной механики.
* Из прикольных особенностей, тут есть Actions Store, в котором есть уже какое-то прописанное множество джоб, которые можно интегрировать в свои пайплайны. То есть если мне нужно добавить pnpm в свой пайплайн в GitLab-CI, то мне надо будет прописать это самостоятельно, или воспользоваться заготовленным docker image с docker hub. В GitHub Actions всё проще, но отстойней, тут инструкции для setup условного pnpm можно стянуть с Actions Store, и он сам всё докачает.
