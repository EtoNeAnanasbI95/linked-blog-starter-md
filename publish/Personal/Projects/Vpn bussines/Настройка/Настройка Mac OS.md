# Настройка и использование
По итогу откроется вот такая штука:
![[telegram-cloud-photo-size-2-5402271064354061346-y.jpg]]
Теперь скопируй вот эту ссылку:
	`vless://69b1991c-264f-42f5-af36-4ccb8287ffc8@195.19.93.131:443?type=tcp&security=reality&pbk=caGAylIGX2IQkx6bTZBqY4AgfsHoUGiClDGQsb6eqGU&fp=chrome&sni=yahoo.com&sid=&spx=%2F&flow=xtls-rprx-vision#main-%40jaimelinapluie-mac`
Нажми правой кнопкой мыши по пустому пространству и выбери Add from clipboard
![[telegram-cloud-photo-size-2-5402271064354061348-y.jpg]]
Дальше всё просто. Когда хочешь vpn, жми на System proxy. Когда нажмёшь макбук на пару секунд подлагнёт, это нормально
![[telegram-cloud-photo-size-2-5402271064354061353-y.jpg]]
Дальше жми на старт, и впн включится
![[telegram-cloud-photo-size-2-5402271064354061357-y.jpg]]
ОЧЕНЬ ВАЖНО: Перед включением VPN включай system proxy, после выключения vpn выключай system proxy
